MIA (Medical Image Analysis): Getting Started

Welcome to MIA, an image editing software developed by Titus-Andronicus. This application can be used for drawing and applying filters to a loaded medical image; additionally, the software supports saving any work done in the application to a PNG or JPG image. This guide contains information regarding Using the Editing Tools, Loading an Image, Modifying Existing Images with Filters, and Saving an Image.

Using the Editing Tools

In the MIA application, there are two tools that can be applied to the canvas, a stamped arrow marker and a red pen. They are selected by clicking the radio button corresponding to their name in the Tool Type of the gray user controls interface. Below is a demonstration of these tools.

Loading an Image

In order to load an image into MIA begin by placing a PNG or JPG file into the MIA directory or navigating to it using the File I/O directory browser and its “../” button. This image should then appear in the Image I/O directory browser. Next, fill the “Image:” text box with the image file’s name; either double click the image from the directory browser menu to auto-fill the box or type the filename including the .jpg or .png extension. Once the box has been filled, click the “Load Canvas” button and the canvas will be automatically resized and replaced with the image file. The undo and redo buttons may be used to undo and redo the load operation. If this image is part of a 3D imaging library or image sequence, MIA will automatically recognize this and enable the Next/Previous buttons to flip between subsequent image. Modifying existing images

Once an image is loaded to the MIA canvas, any tool or filter may be applied to it. Filters range from the Sharpen and Edge Detect to the Threshold, RGB, Quantize, and Grayscale. The options for these may be found in the “Filters” section of the control panel. Some filters can be given scaling values and filters may be applied to the image in succession. Saving an Image

At any time during the program’s operation, the canvas state with any user modifications can be saved to either a JPEG or PNG file. To do this one must simply will the “Image:” textbox with a name for the image to be saved including the .jpg or .png file extension. The new image should then appear in the directory browser of the Image I/O and may be manipulated as any other image.
