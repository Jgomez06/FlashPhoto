Converting and Segmenting a CT Scan Using Mia

Welcome! This tutorial will walk you through on how to convert a set of brain images into a new set where each image has four distinct tissue types visible. Although this may seem like a daunting task, Mia is a powerful tool that supports these types of conversions with a few easy commands.

The first step to navigate to the directory that contains your Mia executable program.

               

             Fig 1. Here we have navigated in the terminal to directory containing mia

Once you have done this, you will need to run your program using four simple parameters:

                                        ./mia inDir -blur 2 -quantize 4 outDir

inDir: The directory that contains the images you would like to process.

-blur 2:  This command blurs the image slightly in order to remove unwanted noise from the picture, giving us a better end result.

Quantize 4: This command separates the image into four distinct colors that allows us to view the distinct tissue types

outDir: This is the name of the directory where the new files will be saved. If the directory does not exists, Mia will conveniently create one for you.

   

                                    Fig. 2. Executing mia takes only 4 simple commands.

That is all! This simple command will now generate a new set of images that will depict four different tissues types. View the images below to see the expected output.
