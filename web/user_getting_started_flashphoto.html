FlashPhoto: Getting Started

Welcome to FlashPhoto, an image editing software developed by Titus-Andronicus. This application can be used for drawing and applying filters to a user-drawn canvas or from any loaded PNG and JPG image; additionally, the software supports saving any work done in the application to a PNG or JPG image. This guide contains information regarding Using the Editing Tools, Loading an Image, Modifying Existing Images with Filters, and Saving an Image.

Using the Editing Tools

In the FlashPhoto application, there are various brushes and tools that can be applied to the canvas, each with a different function and properties. They are selected by clicking the radio button corresponding to their name in the Tool Type of the gray user controls interface. For many of the tools, excluding the eraser, stamp, and blur tool, the color can be changed using the Tool Color box. Below is a demonstration of these tools with different colors applied to each one. The undo and redo buttons may be used to undo and redo any marks made with these tools.

In addition to these simple tools illustrated above (Pen, Spray Can, Caligraphy Pen, Eraser and Highlighter), there is an image-based stamping tool and a blur tool. These will be detailed in the section Modifying Existing Images.

Loading an Image

In order to load an image into FlashPhoto begin by placing a PNG or JPG file into the FlashPhoto directory or navigating to it using the File I/O directory browser and its “../” button. This image should then appear in the Image I/O directory browser. Next, fill the “Image:” text box with the image file’s name; either double click the image from the directory browser menu to auto-fill the box or type the filename including the .jpg or .png extension. Once the box has been filled, click the “Load Canvas” button and the canvas will be automatically resized and replaced with the image file. In addition, the same process may be repeated to activate the image as a stamp. To do this, select the “Stamp” tool from the “Tool Type” box, fill in the image selection box as before, and click the “Load Stamp” button. Now the image may be stamped to the canvas with mouse clicks. The undo and redo buttons may be used to undo and redo the load operation as well as the stamps. Modifying Existing Images

Once an image is loaded to the FlashPhoto canvas, any tool or filter may be applied to it. In addition to the tools covered previous sections, the use of the blur tool becomes more relevant - it may be activated simply by selecting it in the “Tool Type” box. More distinctively however, a number of filters may be applied to the image canvas. These range from the Blur, Motion Blur, and Sharpen to the Sepia and Edge Detect filters. The options for these may be found in the “Filters” section of the control panel. Some filters can be given scaling values and filters may be applied to the image in succession. Saving an Image

At any time during the program’s operation, the canvas state with any user modifications can be saved to either a JPEG or PNG file. To do this one must simply will the “Image:” textbox with a name for the image to be saved including the .jpg or .png file extension. The new image should then appear in the directory browser of the Image I/O and may be manipulated as any other image.
